# Workflow Overview

This project demonstrates an Agent-assisted workflow for engineering research. The workflow is designed for complex modelling tasks that require repeated cycles of analysis, validation, documentation, and manuscript preparation.

## Workflow Stages

1. **Research goal interpretation**  
   The Agent helps clarify the research objective and convert a broad problem into a structured task list.

2. **Task decomposition**  
   The workflow separates the research process into modelling, computation, visualization, verification, and writing tasks.

3. **Model and assumption organization**  
   The Agent helps organize physical assumptions, modelling logic, validation criteria, and revision priorities.

4. **Data-analysis coordination**  
   The Agent coordinates analysis steps and helps keep outputs consistent with the stated research objective.

5. **Figure and table generation**  
   The workflow supports the preparation of publication-oriented figures, tables, and result summaries.

6. **Manuscript update**  
   The Agent assists in converting computational findings into structured technical writing, captions, and revision notes.

7. **Consistency checking**  
   The workflow checks whether figures, tables, text descriptions, and modelling assumptions are mutually consistent.

8. **Public-safe run-log generation**  
   The final step produces a sanitized log that records the workflow without exposing private paths, raw data, unpublished details, or project-specific confidential values.

## Human-in-the-loop Principle

The Agent does not replace the researcher. It supports repetitive, structured, and documentation-heavy tasks, while scientific judgement remains under human control.

## Output Types

The workflow can produce:

- structured research logs;
- model-update summaries;
- figure-generation plans;
- manuscript revision suggestions;
- public-safe documentation for project reporting.
