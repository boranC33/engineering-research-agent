# Privacy and Sanitization

This repository is a public demonstration of an Agent-assisted research workflow. It intentionally focuses on workflow logic rather than private research materials.

## Excluded Information

The following information is intentionally excluded:

- local file paths;
- raw simulation or experimental data;
- unpublished full manuscript content;
- private numerical parameters;
- personal or institutional sensitive information;
- complete project-specific source code;
- confidential intermediate outputs.

## Public-safe Documentation

The public-safe run log keeps the high-level research process, including objectives, decisions, workflow stages, troubleshooting categories, and final outcomes. It removes details that could expose private data, unpublished results, or local project structure.

## Intended Purpose

The purpose of this repository is to demonstrate how an AI Agent can support engineering research workflows while maintaining responsible handling of sensitive research information.
