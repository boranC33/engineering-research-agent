# Sample Manuscript Update

This example illustrates how the Agent can summarize a manuscript-oriented update without revealing confidential technical details.

## Update Goal

Improve the consistency between the modelling section, generated figures, comparison logic, and final interpretation.

## Revision Actions

- clarified the modelling assumptions;
- reorganized the comparison logic;
- aligned figure captions with the manuscript text;
- summarized the final workflow outcome;
- recorded remaining non-critical issues for future review.

## Review Note

All scientific claims should be reviewed by the researcher before use in a formal manuscript.
